# Infra conventions

## Writing style

Capitalize the first letter of every word in headings, bullets, button labels, and
prose — unless the term is canonically lowercase (e.g. `cico-api`, `cloudflared`,
`pnpm`, `shadcn/ui`, env-var keys, file paths, code identifiers). When in doubt,
prefer capitalized.

Examples
- "Add Public Hostname" — not "add public hostname"
- "Cloudflare Pages Env" — not "cloudflare pages env"
- "Restart `cico-api`" — `cico-api` stays lowercase (canonical name)
- "Run `pnpm install`" — `pnpm` stays lowercase
