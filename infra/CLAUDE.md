# Infra conventions

## Writing style

Sentence case. Capitalize the first letter of each sentence. When a word
stands alone (heading, button label, single-word bullet), capitalize it.
Inside a sentence, lowercase the rest unless the term is a proper noun or
canonically lowercase (e.g. `cico-api`, `cloudflared`, `pnpm`, `shadcn/ui`,
env-var keys, file paths, code identifiers).

Examples
- "Restart" — single word, capitalized
- "Add public hostname" — sentence case, only first letter capitalized
- "Restart `cico-api` after the env change" — first letter capitalized, code identifier stays lowercase
- "Run `pnpm install`" — first letter capitalized, `pnpm` stays lowercase
